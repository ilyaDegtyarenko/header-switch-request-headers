# Privacy Policy

Header Switch: Request Headers does not collect, transmit, sell, or share user data.

Header rules, enabled states, and URL filter settings are stored locally in Chrome using `chrome.storage.local`. This data stays on the user's device and is not sent to any external server.

The extension uses Chrome's `declarativeNetRequest` API only to apply request header rules configured by the user.

The extension does not use remote code, analytics, tracking, advertising, or external servers.
